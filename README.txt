Projeto de Algoritmo de Traçado de Raios Não Recursivo (Ray Cast)

Para executar, basta rodar o comando node index.js
Antes disso, é preciso instalar as dependências, com o comando npm install

Todas as imagens geradas estão na pasta images!

A pasta util contém o código relativo ao modelo de Phong e também algumas funções úteis de álgebra linear. O arquivo config.js contém os parâmetros para rodar o programa e gerar as imagens.
